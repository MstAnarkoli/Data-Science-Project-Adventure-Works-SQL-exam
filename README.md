# Project Title: Sales data Analysis of AdventureWOrks 2025 database with SQL and Python

## Project Overview

As a data analyst I have access to business data from the "AdventureWorks 2025" database.  The research needs to use SQL, Python and present results in a Jupyter Notebook. 
The goal is to discover meaningful insights and build visualizations that help to understand the underlying sales trends and patterns.



## Language, Libraries and server
SQL, Python version 3.13.7

numpy, pandas, matplotlib, Jupyter Notebook, sqlalchemy, urllib.parse

MS SQL server

## How to run
_Have python version 3.13.7_

**Clone the repository**:  git clone https://github.com/MstAnarkoli/Data-Science-Project-Adventure-Works-SQL-exam
**Install dependencies** : pip install -r requirements.txt

**Open notebooks/report.ipynb** and run all cells to see the results and charts and interpretations
